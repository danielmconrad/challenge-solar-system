Challenge: Solar System Simulator
=================================

A quick RubyQuiz challenge written in Node.js
See exercise details at [RubyQuiz](http://rubyquiz.strd6.com/quizzes/227-solar-system)

## Usage

Browse to `http://<server-address>/2015-05-02`
